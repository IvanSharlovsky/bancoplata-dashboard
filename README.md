# Engineering Delivery Analytics Dashboard

**Data Analyst Pet Project | Ivan Sharlovskii**

## Overview
Interactive dashboard analyzing software delivery performance and engineering operations using DORA framework metrics.

## Key Features
- **Cycle Time & Lead Time Analysis**: Tracking delivery bottlenecks over time
- **Resource Allocation**: Work distribution by team and priority
- **Anomaly Detection**: Identifying tasks with Cycle Time >14 days
- **Interactive Filters**: Slice by team and priority

## Tools
- Power BI (DAX, data modeling)
- SQL, Python (data generation)
- DORA framework metrics
